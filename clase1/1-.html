HTML (HyperText Markup Language) es el lenguaje principal utilizado para crear y estructurar páginas web. Define la estructura básica de una página web mediante etiquetas que indican cómo debe mostrarse el contenido en un navegador. Las etiquetas HTML son la base de cualquier sitio web y se usan para definir títulos, párrafos, enlaces, imágenes, listas, tablas, formularios, y mucho más.
